Solusi komprehensif untuk interaksi dinamis, SPA routing, forms computation like excel, dan banyak lagi. Tingkatkan pengalaman pengguna dengan mudah tanpa kompleksitas JavaScript yang berlebihan.

Instalasi

Mulai menggunakan LiveDomJS dalam hitungan menit

1 Via Composer (Laravel)

Jika menggunakan Laravel, instalasi LiveDomJs dapat dilakukan langsung melalui Packagist:

# Install package
composer require gadingrengga/livedomjs

# Jalankan instalasi untuk publish asset dan route
php artisan livedomjs:install

2 Penggunaan di Blade

Tambahkan script berikut ke dalam layout Blade Anda setelah jQuery: 

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<script src="{{ asset('vendor/livedomjs/livedom.js') }}"></script>

Catatan: Pastikan jQuery sudah dimuat sebelum LiveDomJS untuk menghindari error.

3 Setup CSRF Token

Tambahkan CSRF token di bagian head template Anda:

<meta name="csrf-token" content="{{ csrf_token() }}">

Dokumentasi Lengkap

Panduan komprehensif untuk semua fitur LiveDomJS

1. AJAX Dinamis

Sistem AJAX otomatis dengan attribute HTML sederhana untuk interaksi server tanpa reload halaman.

Attribute Utama:

  • live-scope - Menentukan controller
  • live-click - Method yang dipanggil saat click
  • live-target - Element target untuk response
  • live-method - HTTP method (GET/POST)
  • live-loading - Tampilkan loading indicator

Event Handlers:

  • live-submit - Submit form via AJAX
  • live-change - Trigger saat value berubah
  • live-keyup - Trigger saat keyup
  • live-input - Trigger saat input
  • live-hover - Trigger saat hover

Contoh Penggunaan:

<div live-scope="user">
    <input name="name" type="text" placeholder="Nama User">
    <button live-click="save" live-target="#result" live-loading="true">
        Simpan User
    </button>
    <div id="result"></div>
</div>

2. Live Computation

Sistem kalkulasi real-time dengan formula Excel-like. Mendukung berbagai fungsi matematika dan tanggal.

Fungsi Matematika

  • sum(field)
  • avg(field)
  • min(field)
  • max(field)
  • count(field)
  • sumif(range, criteria, sum_range)

Fungsi Tanggal

  • rangeDate(start, end)
  • rangeMonth(start, end)
  • rangeYear(start, end)
  • rangeWeek(start, end)

Format Output

  • idr - Format Rupiah
  • currency - Format Dollar
  • decimal - 2 Desimal
  • percent - Persentase
  • days/months/years

Contoh Penggunaan:

<!-- Kalkulasi otomatis -->
<input name="qty" type="number" value="10">
<input name="price" type="number" value="5000">
<input live-compute="qty * price" live-compute-format="idr" readonly>

<!-- Dengan kontrol manual -->
<input name="subtotal" live-compute="sum(rows_?_qty * rows_?_price)" 
       live-compute-format="idr" live-compute-skip="true">

3. Live Conditionals

Kontrol visibility dan styling element berdasarkan kondisi input secara real-time.

Directive yang Tersedia:

  • live-show

    Menampilkan/menyembunyikan element berdasarkan kondisi

  • live-class

    Menambah/menghapus CSS class secara dinamis

  • live-style

    Mengubah inline style secara dinamis

  • live-attr

    Mengubah attribute element secara dinamis

Contoh Kondisi:

<!-- Show berdasarkan value -->
<div live-show="status == 'active'">
    Content untuk status aktif
</div>

<!-- Class berdasarkan kondisi -->
<div live-class="price > 1000 ? 'text-red-500' : 'text-green-500'">
    Harga berubah warna
</div>

4. SPA Router

Single Page Application dengan routing otomatis dan navigasi smooth tanpa reload halaman.

Cara Kerja:

  • â€¢ Intercept semua link dan form dalam region SPA
  • â€¢ Loading content via AJAX tanpa reload
  • â€¢ Update browser history otomatis
  • â€¢ Support browser back/forward button
  • â€¢ Error handling terintegrasi

Konfigurasi:

window.liveDomConfig = {
    spaExcludePrefixes: [
        '/api/',
        '/download/',
        '/admin/export'
    ]
};

Setup HTML:

<!-- Layout dengan SPA regions -->
<div live-spa-region="header">
    <nav>...</nav>
</div>

<div live-spa-region="main">
    <!-- Konten utama yang akan di-update -->
</div>

<div live-spa-region="sidebar">
    <!-- Sidebar yang bisa diupdate -->
</div>

5. Live Data Binding

Sinkronisasi data otomatis antar element dengan two-way data binding yang mudah digunakan.

Fitur Binding:

  • â€¢ Sync otomatis berdasarkan name attribute
  • â€¢ Support input, select, textarea
  • â€¢ Two-way binding dengan live-bind
  • â€¢ Event trigger otomatis (input, change)
  • â€¢ Cross-element synchronization

Contoh Implementasi:

<!-- Input source -->
<input name="username" type="text">

<!-- Elemen yang akan ter-sync -->
<span live-bind="username"></span>
<input live-bind="username" readonly>

6. Auto Polling

Refresh konten secara berkala otomatis. Ideal untuk dashboard real-time, notifikasi, dan monitoring.

Cara Kerja:

  • â€¢ Interval dalam milliseconds
  • â€¢ Auto-abort request sebelumnya
  • â€¢ Error handling otomatis
  • â€¢ Memory management yang baik
  • â€¢ Stop polling saat element dihapus

Contoh Penggunaan:

<!-- Polling setiap 5 detik -->
<div id="notifications" 
     live-poll="5000" 
     live-scope="dashboard" 
     live-click="getNotifications">
    <!-- Konten akan diupdate otomatis -->
</div>

        </button>

    <div id="user-list">
        @include('users.list')
    </div>
</div>

Controller Laravel:

class UserController extends Controller
{
    public function store(Request $request)
    {
        $user = User::create($request->all());
        
        return view('users.list')->render();
    }
}

Kalkulator Live

HTML Form:

    <input name="qty" type="number" value="1">
    <input name="price" type="number" value="100000">

           live-compute="subtotal * 0.1"
           live-compute-format="idr" readonly>
    
    <!-- Total -->
    <input name="total" 
           live-compute="subtotal + tax"
           live-compute-format="idr" readonly>
</div>

Fitur Lanjutan

Kemampuan tingkat lanjut untuk aplikasi yang kompleks

Callback Functions

Jalankan fungsi sebelum dan sesudah AJAX request untuk kontrol yang lebih detail.

<button live-click="save"
        live-callback-before="beforeSave"
        live-callback-after="afterSave">
    Simpan Data
</button>

function beforeSave(element) {
    // Validasi atau konfirmasi
    return confirm('Yakin mau simpan?');
}

Advanced Targeting

Target element dengan selector jQuery yang fleksibel untuk manipulasi DOM yang presisi.

<!-- Target multiple dengan selector -->
<button live-click="update"
        live-target="closest(.card), next(.info)">
    Update
</button>

<!-- DOM actions yang beragam -->
<button live-click="add"
        live-dom="append,show"
        live-target="#list,#notification">
    Tambah Item
</button>

Method dengan Argumen

Kirim argumen spesifik ke controller method untuk logika yang lebih dinamis.

HTML:

<!-- Single argument -->
<button live-click="updateStatus('active')">
    Aktifkan
</button>

<!-- Multiple arguments -->
<button live-click="setPrice(100000, 'idr')">
    Set Harga
</button>

<!-- Dynamic dengan element value -->
<button live-click="calculate(this.dataset.id, qty)">
    Hitung
</button>

Controller:

public function updateStatus($status)
{
    // $status berisi 'active'
    $model->status = $status;
    $model->save();
}

public function setPrice($amount, $currency)
{
    // Handle price dengan currency
    return response()->json([
        'success' => true,
        'data' => "Price set to {$amount} {$currency}"
    ]);
}
